# 实现资金同步按钮与进度条逻辑

## 1. 更新前端 API Hook
- 修改 `frontend/src/hooks/useWailsAPI.ts`，添加 `StartFullMarketSync` 方法的封装，确保类型安全。

## 2. 创建 `MoneyFlowSyncButton` 组件
- **文件路径**: `frontend/src/components/MoneyFlowSyncButton.tsx`
- **UI 设计**:
    - 一个带图标的按钮（使用 `Database` 或 `RefreshCw` 图标）。
    - 按钮右侧或下方显示进度条和文本信息。
- **状态管理**:
    - `syncing`: 是否正在同步。
    - `progress`: 包含 `current`, `total`, `currentStock` 等信息。
- **事件监听**:
    - 监听 `sync_progress` 事件，实时更新进度状态。
    - 进度条逻辑：计算百分比 `(current / total) * 100`。
- **交互逻辑**:
    - 点击按钮调用 `StartFullMarketSync`。
    - 同步过程中禁用按钮，显示 Loading 状态。
    - 同步完成后显示成功提示。

## 3. 集成到 `DecisionPioneerPage`
- 在页面顶部工具栏区域（`ScanButton` 旁）引入并放置 `MoneyFlowSyncButton`。
- 调整布局，确保两个按钮并排显示且风格统一。

## 4. 样式调整
- 使用 Tailwind CSS 实现毛玻璃效果和进度条动画。
- 确保进度条颜色与主题（深色模式）协调。
